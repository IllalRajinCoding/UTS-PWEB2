# INSTALL PROJECT

## Installation Steps

1. **Install Tailwind CSS via npm**
   ```bash
   npm install tailwindcss @tailwindcss/cli
   ```
2. **Run Tailwindcss**
   Run diterminal untuk memproses tailwind
   ```bash
   npm run build
   ```
3. **Kejar waktu bang**
   ```goks
Yo semangat bang
   ```
